Opensource Projects

Opensource projects I developed or take part in

  • SatNOGS Satellite Network of Ground Stations

    This project uses raspberry pis and RTLSDRs to create a global network of ground stations for lsitening to satelites in oribt.

    I run a ground station for this project and also contribute to this project in to ways

  • Realtime Satnogs Network Activity Map

    This project uses the api from the Network site of the SatNOGS project to create a map showing what sattellites are being actively listened to and what stations are listening to them.

    The code for the majority of this was writen in December of 2018 with major changes happening around Feburary of 2019.

    The current source code is available from a Gitlab Repository

    I am currently working on a new version of the website using Django instead of Flask as the backend. It has its own Gitlab Repositoy.

  • Satnogs Network Activity Replay Map

    This project uses the api from the Network site of the SatNOGS project to create a map showing a replay of an event for a single satellite on the network.

    This started as a neat way to reply Internation Space Station Slow Scan TV events. (ISS SSTV) But has evolved to allow replaying any satellite on the network.

    This project is not currently being developed or mantainted due to how sloppy the code currently is. Its current code is available from a Gitlab Repositorty

    I will eventually be merging this into the new Realtime Satnogs Network Activity Map. This is expected to happen far in the future.
